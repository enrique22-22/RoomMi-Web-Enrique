# RoomMi-Web-Enrique

Landing de presentación para **RoomMi**, aplicación de alquiler compartido de habitaciones (proyecto intermodular 1º DAM · Enrique Soriano).

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Producción

```bash
npm run build
npm start
```

Despliega en [Vercel](https://vercel.com) para obtener un enlace público.

## Contenido

La landing explica el problema, la solución, funcionalidades, perfiles de usuario y lista de espera.
